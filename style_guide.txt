# LATEX GENERATION STYLE GUIDE
# Goal: Academic CV (IEEE Style)
# Font: Times New Roman

[DOCUMENT SETUP]
- Document Class: \documentclass[11pt]{article}
- Package: \usepackage{mathptmx} % Sets font to Times New Roman
- Package: \usepackage[margin=0.75in]{geometry}
- Package: \usepackage{enumitem} % For list customization
- Package: \usepackage{titlesec} % For section formatting
- Package: \usepackage{hyperref} % For links
- Package: \usepackage{lastpage, fancyhdr} % Optional for page numbering
- Layout: Single column

[FONT SIZING & TYPOGRAPHY]
- Base Font: 11pt (set in documentclass)
- Title (Name): 16pt Bold -> Use \fontsize{16}{19}\selectfont \textbf{Name}
- Section Headers: 12pt Bold -> Use \large \textbf{SECTION NAME}
- Body Text: 11pt (Standard)

[HEADER SECTION]
- Alignment: Centered
- Content:
  1. Name (16pt, Bold)
  2. Contact Line (11pt): City, State | email | phone | linkedin_url
- Spacing: \vspace{0.5em} after contact info.

[SECTION FORMATTING]
- Command: \titleformat{\section}
- Style: \large\bfseries\uppercase
- Decoration: \titlerule (Horizontal rule below text)
- Spacing: \titlespacing*{\section}{0pt}{12pt}{6pt} (Adjust for visual breathing room)

[ENTRY FORMAT]
# Used for Education, Experience, Projects
- Structure:
  \noindent \textbf{Title/Role} \hfill \textbf{Date Range} \\
  \textit{Institution/Company} \hfill \textit{Location (Optional)}
- Details:
  \begin{itemize}[leftmargin=*, noitemsep, topsep=0pt]
    \item Detail 1
    \item Detail 2
  \end{itemize}
- Spacing: \vspace{0.5em} between distinct entries.

[ACADEMIC/IEEE CONVENTIONS]
- Dates: Use "May 2025" or "2023--Present" (use en-dash `--` for ranges).
- Citations: If including a publication list, use standard IEEE format (e.g., [1] J. Doe, "Title," Journal...).
- Tense: Past tense for past roles, Present tense for current roles.

[SKILLS SECTION]
- Format: \noindent \textbf{Category Name:} Skill 1, Skill 2, Skill 3 \\
- Grouping: Keep relevant keywords together (e.g., Software, Hardware, Languages).
